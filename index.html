BMI calculation development

Jing WU

Development objectives

  1. Calculate the BMI index according to the input values (weight & height)

  2. Re-calculate the BMI when the input changing

Example of calculating BMI

We assume Jack has weight (60 kg), and height (1.65 m). So Jack's BMI is

weight (kg) / (height (m) * height (m))
